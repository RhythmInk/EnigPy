EnigPy

An enigma machine emulator written in Python 3 and Flask.

Enigma Machine

Instructions for encrypting text will appear here. Once the user enters the mssage and presses the button this text will be replaced with the encrypted cypher text.

Enigma Machine

Instructions for decrypting text will appear here. Once the user enters the mssage and presses the button this text will be replaced with the decrypted plain text.
